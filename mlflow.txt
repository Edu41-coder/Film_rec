pip install mlflow


Pour tester:

Démarrer le serveur MLflow :
mlflow ui

Exécuter l'script
Dans une nouvelle fenêtre PowerShell exécuter votre script Python:
python Models/train_model.py

Accéder à l'interface MLflow :
Ouvrez un navigateur web et allez à l'adresse suivante :
http://127.0.0.1:5000

mettre au debut su script python apres les importations
mlflow.set_tracking_uri('http://localhost:5000')

Intégrer le versioning du modèle avec MLflow
Créer le fichier dvc.yaml
Adapter le script Python :
l'script train_model.py doit être configuré pour utiliser MLflow pour le versioning du modèle
Utiliser DVC pour gérer le pipeline :
Avec le fichier dvc.yaml en place et le script Python configuré pour utiliser MLflow, 
on peut maintenant utiliser DVC pour exécuter votre pipeline. 
La commande dvc repro exécutera le script d'entraînement, 
qui à son tour enregistrera le modèle et les métriques dans MLflow.
